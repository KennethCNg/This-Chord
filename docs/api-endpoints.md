# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `GET /api/users`
- `GET /api/users/:id`
- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Messages

- `GET /api/messages`
- `POST /api/messages`
- `DELETE /api/messages/:id`

### Channels

- `GET /api/channels`
- `POST /api/channels`
- `GET /api/channels/:id`
- `UPDATE /api/channels/:id`
- `DELETE /api/channels/:id`

## BONUS
### Servers

- `GET /api/servers`
- `POST /api/servers`
- `GET /api/servers/:id`
- `UPDATE /api/servers/:id`
- `DELETE /api/servers/:id`
